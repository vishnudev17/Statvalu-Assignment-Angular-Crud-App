Employee Task Dashboard

A simple Angular app to manage employee tasks. It lets you add, edit, delete, filter, and sort tasks. All data is saved in LocalStorage so it stays even after refreshing.

Features

Add, edit, and delete tasks

Search by task title

Filter by status (Pending / In-Progress / Completed)

Sort by due date or priority

Pagination

Reusable components:

Input

Dropdown

Status badge

Stores all tasks in LocalStorage

Pages
Task List Page

Shows all tasks

Filter, search, sort, and delete tasks

Button to add a new task

Add/Edit Task Page

Fields:

Title

Description

Employee

Priority

Due Date

Status
